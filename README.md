# NovaCart

Frontend-only e-commerce application built with Vite, React, React Router, and Tailwind CSS. Authentication, users, cart, wishlist, and theme preference are simulated with `localStorage`.

## Run

```bash
npm install
npm run dev
```

## Demo Features

- Product listing, search, category filter, and price sorting
- Product details page
- Cart with quantity updates and persisted totals
- Mock registration/login/logout with duplicate-email prevention
- Protected account route
- Wishlist toggle
- Dark mode toggle
- Lightweight toast notifications
